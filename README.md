
# Proyecto Semáforo

## Memoria descriptiva
El sistema simula el funcionamiento de un semáforo típico de tráfico, con tres estados
principales:

### Duración de Estados:
     Verde: 30 segundos
     Amarillo: 10 segundos
     Rojo: 30 segundos

### Transiciones:
    Verde → Amarillo
    Amarillo → Rojo
    Rojo → Amarillo
    Amarillo → Verde
